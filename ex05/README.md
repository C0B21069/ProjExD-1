### ゲーム概要
- ex05/fight_kokaton.pyを実行すると，1600x900のスクリーンに草原が描画され，こうかとんを移動させ飛び回る爆弾から逃げるゲーム

- 矢印キーでこうかとんを上下左右に移動する
