# 第6回
## アクションゲーム
### ゲーム概要
- を実行すると，800x480のスクリーンに草原が描画され，こうかとんをゴールまで運ぶ
- こうかとんがゴールポールの触るとクリア
- クリアするとクリア画像が表示される
### 操作方法
- 矢印キーでこうかとんを上下に移動する
- スペースキーでこうかとんがジャンプする
### クリア条件
- ブロックなどの障害物を乗り越えてこうかとんゴールポールに到達したらクリア！！！
